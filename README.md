# ESP32 Distance Sensor Alarm 🚨

A simple obstacle detection module for my autonomous car project. It uses an HC-SR04 ultrasonic sensor and an active buzzer. If an object is closer than 20 cm, the alarm triggers automatically.

## Hardware 
* ESP32 (Wroom)
* HC-SR04 Ultrasonic Distance Sensor
* Active Buzzer

## Wiring 
**HC-SR04 Trig**  GPIO 5 

**HC-SR04 Echo**  GPIO 18 

**Active Buzzer**  GPIO 19 

**VCC**  5V / Vin
